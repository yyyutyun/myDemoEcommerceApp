
# My Demo project


## Intro
This is a demo project that is a small-scale Eshop to demostrate the capability of FULL-STACK development.



## Language involved
1）FrontEnd： React Native

2）Backend： NodeJS



## Detail
1） Frontend：

-Expo

-React Navigation

-JWT decoder

-Redux and Context API

-Styled-components

-Native Base

2） Backend：

-Express

-Mongoose

-JsonWebToken
